Password Security Tips

  • Use long passwords - Aim for at least 12 characters. Longer passwords are generally more secure.
  • Use a mix of characters - Combine uppercase letters, lowercase letters, numbers, and symbols.
  • Avoid personal information - Don't use names, birthdates, or common words that could be easily guessed.
  • Use different passwords - Each account should have a unique password to prevent multiple breaches if one password is compromised.
  • Use a password manager - Store your complex passwords securely in a reputable password manager.
  • Enable two-factor authentication - When available, use 2FA for an extra layer of security beyond just your password.
